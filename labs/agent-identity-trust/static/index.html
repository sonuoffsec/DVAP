Identity Spoofing

Agent identity is passed as the X-Agent-Id header. The authorization service trusts this header without cryptographic verification.

Available Endpoints
/api/resources/public
/api/resources/private (admin-agent only)
/api/admin (admin-agent only)
Challenge 1: Change the X-Agent-Id header to "admin-agent" and access /api/resources/private. The server trusts the identity header without verification.
